## 프로젝트 목적
- react, redux, redux-thunk, styled-component를 종합적으로 활용해 보기 위함으로 제작한 프로젝트


## watcham 프로젝트 소개
- 20개의 최신 인기 영화에 대한 정보를 제공


## 기능
- 20개의 최신 영화 리스트업
- 검색창을 통한 검색기능
- 영화 카드 클릭 시 영화 상세 내용 제공
- 좋아요 클릭을 통한 영화 찜기능


## Stack
`HTML` `Javascript` `CSS` `React` `Styled-component`
